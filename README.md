# Rushi Shah - Professional Portfolio ("Newsprint" Edition)

A meticulously designed, high-performance web portfolio built to showcase the professional identity of Rushi Shah. Characterized by a "Newsprint" design system, this portfolio emphasizes stark, zero-radius geometry, high-contrast typography, and a journalism-inspired editorial layout.

## ✨ Key Features
- **Newsprint Design System**: A strict, zero-radius aesthetic using `EB Garamond` for editorial authority and `Google Sans Flex` for modern interface clarity.
- **Dynamic Market Ticker**: A live-simulated terminal ticker providing a unique top-level narrative.
- **Custom Digital Identity**: Includes a highly customized, pixel-perfect LinkedIn Profile Card integration.
- **Multi-Page Architecture**: Fully client-side routed architecture featuring:
  - `Home`: The front-page breaking news experience.
  - `About`, `Projects`, `Case Decks`, `Achievements`: Deep-dive editorial content.
  - `Resumes`: A curated dossier section for specialized resumes (AI/ML, Finance, Consulting).
  - `Contact`: Professional coordinates and digital identity linking.
- **Custom Cursor**: A global, precision-engineered cursor integrated directly into the CSS engine.

## 🛠 Tech Stack
- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (CSS-first architecture)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Package Manager**: [Bun](https://bun.sh/)

## 🚀 Quick Start (Local Development)

1. **Clone the repository**
   ```bash
   git clone https://github.com/RushishahOG/Rushi_Shah_Portfolio.git
   cd Rushi_Shah_Portfolio
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start the development server**
   ```bash
   bun run dev
   ```
   Open `http://localhost:5173` to view it in your browser.

## 🌍 Deployment
This project is optimized for deployment on **Vercel**.
It includes a `vercel.json` configuration file that automatically handles SPA (Single Page Application) routing rewrites.

1. Connect this repository to your Vercel account.
2. Ensure the framework preset is set to **Vite**.
3. Deploy! Vercel will automatically route all traffic seamlessly to `index.html`.
